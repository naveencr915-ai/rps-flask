\# Rock Paper Scissors – Web Game 🎮



\## 📌 Project Overview

A responsive Rock Paper Scissors game built using HTML, CSS, and JavaScript.  

The game runs directly in the browser and provides an interactive user experience with simple game logic and smooth UI flow.



This project was created to strengthen core frontend development skills and understand DOM manipulation, event handling, and basic game logic implementation.



\---



\## 🚀 Features

\- Classic Rock Paper Scissors gameplay

\- User vs Computer mode

\- Randomized computer selection

\- Instant result calculation (Win / Lose / Draw)

\- Simple and responsive UI design

\- Smooth and interactive user experience

\- Lightweight and fast browser execution



\---



\## 🛠️ Tech Stack

\- HTML5 – Structure of the game

\- CSS3 – Styling and layout design

\- JavaScript (Vanilla) – Game logic and interactivity



\---



\## 🎮 How to Play

1\. Open the game in a browser

2\. Click on Rock, Paper, or Scissors

3\. The computer automatically selects its move

4\. The result is displayed instantly:

&#x20;  - Win

&#x20;  - Lose

&#x20;  - Draw



\---



\## ⚙️ Game Logic

\- Rock beats Scissors

\- Scissors beats Paper

\- Paper beats Rock

\- Same choices result in a Draw



The computer choice is generated using random selection in JavaScript.
